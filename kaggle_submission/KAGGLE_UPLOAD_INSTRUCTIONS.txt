HOW TO UPLOAD TO KAGGLE
=======================

STEP 1: CREATE DATASET
1. Go to https://kaggle.com → Create → Dataset
2. Click "Upload Files"
3. Upload these files:
   - emotions_config.csv
   - meals_database.csv
   - interview_questions.json
   - README.md
   - DATASET_DESCRIPTION.txt

4. Fill in:
   - Title: "Mood-to-Meal Butler: Emotion Detection & Meal Recommendation"
   - Description: (copy from DATASET_DESCRIPTION.txt)
   - License: Creative Commons Attribution 4.0 International
   - Tags: emotion-detection, meal-recommendation, ai, nlp, food-tech

5. Click "Create"

STEP 2: CREATE NOTEBOOK
1. Go to https://kaggle.com → Create → Notebook
2. Click "+ Code"
3. Copy contents from kaggle_notebook.py
4. Add markdown cells for sections

5. Link dataset:
   - Click "Add input"
   - Select your dataset you just created
   - Code will reference: /kaggle/input/your-dataset-name/

6. Run notebook to show demo output

7. Publish as "Public" (so judges can see)

STEP 3: ADD TO SUBMISSION
1. In notebook, go to Share → Add to Competition
2. Select relevant competition
3. Write description (2-3 sentences about your project)
4. Submit
