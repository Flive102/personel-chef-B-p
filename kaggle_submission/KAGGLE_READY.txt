- Click "Create"

STEP 3: Note your dataset path
- After creating, Kaggle shows: /kaggle/input/your-dataset-name/
- Copy this path

STEP 4: Create Kaggle Notebook
- Go to kaggle.com → Create → Notebook
- Click "+ Code"
- Copy kaggle_notebook.py content
- Replace '/kaggle/input/mood-to-meal-butler-kaggle/' with your actual dataset path

STEP 5: Link Dataset to Notebook
- In notebook, click "Add input"
- Select your dataset
- Run cells
- Verify outputs show emotion detection working

STEP 6: Publish & Submit
- Click "Publish"
- Set to "Public" (judges need to see)
- Go to competition page
- Click "Submit" or "Add notebook"
