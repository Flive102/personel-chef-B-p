MOOD-TO-MEAL BUTLER: KAGGLE DATASET

Dataset Description
===================

This dataset powers "Mood-to-Meal Butler" - an AI system that detects emotions and recommends personalized meals based on emotional state, preferences, and dietary needs.

The dataset contains three core components:

1. EMOTIONS_CONFIG.CSV
   - 26 emotion categories with auto-detection keywords
   - Keywords for natural language emotion recognition
   - Meal filter recommendations for each emotion
   - Examples: sadness, exhaustion, joy, stress, anger, fear, love, hope

2. MEALS_DATABASE.CSV
   - 40 sample meals (274 total in production)
   - Multilingual names (English + Vietnamese)
   - Mood associations (which emotions benefit from which foods)
   - Nutritional characteristics and health tags
   - Time requirements, budget levels, cuisine types
   - Energy levels and descriptions

3. INTERVIEW_QUESTIONS.JSON
   - 6 personalized interview questions
   - Bilingual support (English + Vietnamese)
   - Options for: mood, cravings, group size, budget, time, diet
   - Structured JSON format for easy parsing

How To Use
==========

1. EMOTION DETECTION
   - Load emotions_config.csv
   - Match user input against keywords
   - Return detected emotion category

2. MEAL RECOMMENDATION
   - Load meals_database.csv
   - Filter by mood_tags matching detected emotion
   - Apply budget/time/diet filters from interview
   - Return 9 (or 3) recommended meals

3. PERSONALIZATION
   - Load interview_questions.json
   - Ask user for preferences
   - Store answers in session
   - Apply as filters to meal recommendations

Example Code (Python)
====================

import pandas as pd

# Load datasets
emotions = pd.read_csv('emotions_config.csv')
meals = pd.read_csv('meals_database.csv')

# Detect emotion
def detect_emotion(text):
    for emotion, keywords in emotions.items():
        if any(kw in text.lower() for kw in keywords.split(',')):
            return emotion
    return None

# Recommend meals
emotion = detect_emotion("i am tired")
recommendations = meals[meals['mood_tags'].str.contains(emotion)]

print(recommendations[['name_en', 'mood_tags', 'description']])
